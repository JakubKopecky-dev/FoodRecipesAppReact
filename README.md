# Food Recipes App (React)

Simple web application for managing cooking recipes.
This project was created as a learning exercise to practice React, TypeScript and MUI.

## Features
- View recipe details
- Add ingredients to a recipe
- Edit existing ingredients
- Delete ingredients
- Display recipe categories, cooking time and rating
- Error handling using Snackbar alerts

## Tech Stack
- React
- TypeScript
- MUI
- React Router
- REST API communication

## Purpose of the project
The goal of this project was to learn the basics of building a modern frontend application using React and working with component state, dialogs, forms and API calls.
